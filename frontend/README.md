# Vue 3 + TypeScript + Vite

This template should help get you started developing with Vue 3 and TypeScript in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about the recommended Project Setup and IDE Support in the [Vue Docs TypeScript Guide](https://vuejs.org/guide/typescript/overview.html#project-setup).



## 產品 - EOP English Admin
英文補教用的管理介面，其中包含管理員使用的後台介面，還有提供給學生使用的預約中心及老師個人預約課管理介面。

## 目標用戶
補教班管理人員、學生、外語老師

## 配色要求
主色（ #0f52ba）+ 強調色 + 輔助色 + 狀態色( success, warning, error, danger, info )

## 頁面結構 - 

### Dashboard
基礎歡迎畫面

### 個人設定
個人資料區塊、LINE帳號綁定區塊、通知設定區塊、關於LINE綁定說明區塊

### 教師管理
提供篩選區塊（關鍵字搜尋、資料狀態選單、約聘類型選單），名片式管理（老師姓名、約聘類型：全職／兼職、電話、email、帳號驗證、啟用停用開關、編輯按鈕、合約管理按鈕），有分頁可篩選每頁n筆資料檢視。

### 學生管理
提供篩選區塊（關鍵字搜尋、資料狀態選單、學生身份選單）及帶有分頁的學生列表，可針對單一學生開啟抽屜或彈窗進行個人資訊編輯，以及檢視購課合約畫面。

### 預約管理
提供篩選區塊（關鍵字搜尋、資料狀態選單、學生選單、教師選單、課程選單、課程狀態選單）、批次按鈕(新增／更新／刪除)、列表以「日曆模式」顯示（可切換「清單模式」），清單模式可針對表格做批次更新動作，日曆模式以「月曆」為單位顯示，點擊單筆可開啟詳細資料抽屜。

#### 課程管理
提供篩選區塊（關鍵字搜尋、資料狀態選單），列表以「卡片模式」顯示（可切換「清單模式」），卡片內包含課程名稱、課程時數、價格、級別、對象等基本資訊。

### 請假管理
提供篩選區塊（資料狀態選單），列表以表格方式顯示，表格欄位包含學生姓名、請假日期、請假時段、請假原因、審核狀態、核准／駁回按鈕。

### 系統設定
    1. 員工管理: 提供篩選區塊（關鍵字搜尋、資料狀態選單、約聘類型選單），表格式管理（員工姓名、約聘類型：全職／兼職、電話、email、帳號驗證、啟用停用開關、編輯按鈕、合約管理按鈕），有分頁可篩選每頁n筆資料檢視。
    2. 系統帳號管理: 提供篩選區塊（關鍵字搜尋、資料狀態選單、角色類型選單），表格式管理（姓名、帳號類型：管理員／老師／學生／員工、電話、email、帳號驗證、啟用停用開關、編輯按鈕、權限設定按鈕），有分頁可篩選每頁n筆資料檢視。
    3. 權限設定: 提供新增角色權限，並可針對員工、系統帳號進行權限設定（可依角色分組），管理已建立之權限設定。 
    4. zoom帳號設定: 提供串連zoom帳號設定與管理介面。

### 學生預約中心
提供篩選區塊（關鍵字搜尋、資料狀態選單、學生選單、教師選單、課程選單、課程狀態選單）、批次按鈕(新增／更新／刪除)、列表以「日曆模式」顯示（可切換「清單模式」），清單模式可針對表格做批次更新動作，日曆模式以「月曆」為單位顯示，點擊單筆可開啟詳細資料抽屜。

### 老師個人預約課管理
提供篩選區塊（關鍵字搜尋、資料狀態選單、學生選單、課程狀態選單）、列表以「日曆模式」顯示（可切換「週模式」），清單模式可針對表格做批次更新動作，日曆模式以「月曆」為單位顯示，點擊單筆可開啟詳細資料抽屜。並可設定可上課時段。
